# Calculadora de Horas Extras

Aplicación web para registrar horas extras, operaciones y gastos. La app utiliza Firebase para autenticación y almacenamiento en tiempo real, y está diseñada con HTML, CSS (Tailwind) y JavaScript.

## Instalación

1. Clonar el repositorio.
2. Ejecutar `npm install` para instalar dependencias (si usás npm).
3. Configurar Firebase en `public/firebase.config.js` con tus credenciales.
4. Ejecutar `npm run start` para arrancar la aplicación (usa Live Server o similar).

## Uso

- Registrar horas extras, operaciones y gastos.
- Visualizar reportes simples y resúmenes.
- Administrar usuarios (funcionalidad de multiusuario y roles).
